# Adobe Softwares

## Install Adobe Softwares on Linux (wine)

https://www.linuxnest.com/how-to-run-photoshop-on-linux-an-ultimate-guide/

#### Dependencies
`allfonts`, `atmlib`, `dotnet48`, `gdiplus`, `vcrun2013`, `vcrun2019`, `vcrun2022`, `msxml3`, `msxml4`, `msxml6`


## Open-source Alternatives (Photoshop/Illustrator)
- https://inkscape.fr/
- https://www.gimp.org/downloads/
- https://krita.org/fr/features/
- https://graphite.art/

### Using the correct format
Ask for/export `.tiff` instead of `.psd` files.
